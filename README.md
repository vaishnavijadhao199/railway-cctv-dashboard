# railway-cctv-dashboard
The IoT CCTV Railway Dashboard is a full-stack security platform for railway surveillance. Built with FastAPI, React, and PostgreSQL, it features real-time camera monitoring, AI-powered alerts, and advanced analytics. It centralizes station security data, streamlining incident response and enhancing passenger safety.

# 🚆 IoT CCTV Railway Monitoring Dashboard

A full-stack IoT monitoring dashboard for railway security systems.  
This system allows centralized monitoring of railway CCTV cameras with AI-powered alerts and analytics.

---

# 📊 Dashboard Features

• Live CCTV monitoring  
• AI detection alerts  
• Video recording management  
• Station analytics dashboard  
• Real-time system monitoring
